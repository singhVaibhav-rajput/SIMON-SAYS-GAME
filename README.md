# SIMON-SAYS-GAME
A simple Simon Says memory game built with HTML, CSS, and JavaScript.
index.html   The main HTML file containing the structure, title, and the four colored buttons.style.
css          Handles the visual styling of the game, including colors, layout, and the  flashing effects (.flash, .userflash).
script.js    Contains all the core JavaScript logic for game state management, sequence generation, input handling, and flow control.

Start: Press any key on your keyboard to initiate the game.

Watch: Observe the sequence of colored buttons that light up.

Repeat: Click the buttons in the exact same order they flashed.

Advance: Successfully repeating the sequence moves you to the next level, where a new color is added to the sequence.

Game Over: If you click the wrong button, the game ends. Your score is displayed as the highest level you reached. Press any key to restart.
